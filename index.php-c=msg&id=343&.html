波动速读
发布时间：2014-08-14   点击率：414

波动速读最早是由日本七田真博士提出的。分为慢波和快波，慢波是1秒看1页书，快波是哗啦啦翻书看完一本书。

那波动速读和快速阅读有什么区别呢？

      快速阅读是指用你现在的理解途径和方法，通过一些技巧上的改善，达到快速阅读的目的。比如，浏览法、鉴别阅读法、扫读法、固定程序阅读法等等。
       波动速读是指重新建立起理解模式，例如无声理解路径或者右脑阅读路径，实现“眼脑直映”的阅读效果。其特点是从理解思维途径、眼睛运用技巧这样的根源上去提高阅读效率。
两者的区别打个比方：
       波动速读是修炼内功。内力提升之后，任何招式都会有强大的威力。快速阅读技巧就是招式，没有内力的支撑，只能是花架子。通过很多朋友训练的经验也可以看出：
       1.有的朋友说，练习完之后，总想着快速阅读、快速阅读，想把眼睛转的更快，看更多的字。但是要命的是，速度就是快不起来。或者勉强快起来了，理解效果却又低的可怜。或者，知道用一些技巧，可以很快的把一篇文章的重点找到，但是对付这些重点的时候，还得一个字一个字的啃。这类的训练经验，就是单纯的练习的快速阅读技巧。
       2. 还有的朋友，训练完之后，感觉理解非常快，眼睛看到字就能理解，大脑运转很快，没有多余的动作。甚至感觉眼睛跟不上大脑的运行，感觉眼睛成了拖累。这类的训练经验，就是练习的波动速读。
        可以肯定一点波动速读不是骗人的，有些人就把一些快速阅读技巧包装一下当作噱头，并大肆吹嘘其效果，加上一些波动速读里面的眼睛训练方法，但是缺乏根本理解途径的训练，导致眼睛训练成了无本之木。
