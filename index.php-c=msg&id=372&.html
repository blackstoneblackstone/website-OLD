英语思维智能学习法
发布时间：2014-08-14   点击率：854

              英语思维智能学习法：

               大脑高级水平的智能化训练，即学生使用右脑对英语课文进行照相，此训练打通了右脑对外语的深层记忆回路。整个训练是培养孩子的内感觉、内视觉、内听觉。训练中学生将英语课文、单词和语音同时输入大脑，文字在脑中像照相一般清晰可见，同时语音在耳边回荡，学生可看着脑中的英语课文和单词背诵、默写，学习英语的兴趣和能力大增，从根本上解决了学英语难的问题。此技能还可用来自学其他外语，文字同样能够输入大脑。

         

           常见问题：
                           1、学习周期多久？英语思维智能学习法是什么意思？
                                 寒暑假 连续 22天训练  每天3小时  。智能学习法，简单讲就是孩子会自主学习英语或者一门别的语言。 
                           2、学习费用多少？多大孩子可以报名学习？
                                  目前活动价是9000元，三年级以上学生均可报名。
                           3、授课老师是谁？经验如何？
                                 授课老师是国翰教育多年的五阶全脑潜能开发训练师，英语训练经验丰富，口语标准，幽默深受孩子喜欢。已经联系训练国翰5期学员。
      
           学生感悟：
                                                  通州三中李同学：

                                     

           

            “学完最后一步后，我掌握了英语课文照相和录音的能力。所有英语课文，我都可以照进大脑里，并可以照大脑里的文字读出来，也可以跟着大脑录制的声音读出来，还可以照大脑里的课文写出来。

             学完国翰右脑潜能开发悟进法后，我掌握了很多以前没有的能力，我觉得这对我的学习有很大的帮助”

        

          家长感悟： 李同学妈妈讲“孩子英语给报过三年辅导班，成绩也可以，但是中途停了辅导班成绩就下来了，孩子不会学对辅导班老师的依赖性太强，为此找到国翰训练班进行训练，老师和环境也不错，感觉国翰的训练方式和方法也非常独特，孩子训练之后英语持续稳步在班级前3名，我现在很开心，感谢你们。。。。”
